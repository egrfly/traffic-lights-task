Your task

WARNING: this task may not be suitable for those with photosensitive epilepsy. Traffic lights may flicker rapidly if the wait command is not used appropriately.

The objective is to mimic, as accurately as possible, a set of UK traffic lights.

For a quick reminder of the light sequence involved, this demonstration video should help.

The following commands are available to you:

  • red off
  • red on
  • amber off
  • amber on
  • green off
  • green on
  • wait x (replace x by the number of seconds you wish to wait for before moving onto the next command)

You should write one command per line. The sequence of commands will repeat until stopped.
